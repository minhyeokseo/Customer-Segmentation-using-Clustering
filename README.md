이 프로젝트는 캐글에서 제공하는 브라질의 이커머스 데이터를 이용하여 다양한 변수로 고객을 k-means알고리즘을 이용하여 군집화 합니다.  
각 군집에 따른 마케팅 전략을 수립하고 의사결정자가 이를 의사결정자가 한 눈에 볼 수 있도록 대시보드화 합니다.
